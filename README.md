# Kodata

**Kodata** est la première communauté dédiée à la data à Madagascar. Elle réunit data analysts, data engineers, data scientists et curieux du numérique pour apprendre, partager et collaborer — quel que soit leur niveau ou leur secteur.

## À propos

Née au sein du **RLC (Regional Leadership Center) Madagascar Chapter**, Kodata est portée par la conviction que la maîtrise de la donnée est un levier stratégique pour le développement de Madagascar.

- **Mission** — Créer un espace inclusif où les passionnés de la data à Madagascar peuvent apprendre, partager et collaborer.
- **Vision** — Faire de Madagascar un acteur de l'économie de la donnée en Afrique.
- **Adhésion** — Gratuite et ouverte à tous.

## Ce dépôt

Ce repo contient le code source du site web de Kodata.

```
kodata/
├── web/        # Site Next.js (App Router)
│   ├── app/    # Pages et routes
│   ├── components/  # Nav, Footer
│   ├── lib/    # Données (membres, projets, événements)
│   └── public/ # Assets (logos, images)
└── docs/       # Documentation du projet
```

## Stack technique

- **Framework** — Next.js 14+ (App Router)
- **Langage** — TypeScript
- **Style** — Tailwind CSS (design brutalist/rétro)
- **Icônes** — Lucide React

## Lancer le projet en local

```bash
cd web
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.
